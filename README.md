# Kartkówka Git - Grupa Poniedziałek 13.15-14:45

## Czas na wykonanie kartkówki 12 min. 

Przygotuj przygotuj swój token oraz adres e-mail do GitHub.  
Skonfiguruj tozsamość: 

```git config --global user.name "John Doe"```

```git config --global user.email johndoe@example.com```


1. Sklonuj to repozytorium na swój komputer.

2. Utwórz nową gałąź (branch) o nazwie: konfiguracja.

3. Na nowej gałęzi utwórz plik o nazwie config.json.

4. Wpisz do pliku config.json dokładny tekst: {"status": "gotowe"}

5. Zrób commit z wiadomością: "Dodano plik konfiguracyjny".

6. Złącz (merge) gałąź konfiguracja z powrotem do gałęzi main.

7. Wypchnij (push) zmiany z gałęzi main na serwer.
